# User Context for Creative

## Available Generation Models (what Producer can execute)
- **Seedream 4.5**: Best image quality, 4K, text-only input — use for hero visuals
- **Seedream 4.0**: Supports reference images, multi-image fusion — use for brand consistency
- **Vidu Q2 Pro**: Best video quality from image — use for final renders
- **Vidu Q2 Turbo**: Fast video from image — use for drafts and iterations
- **Vidu Q2 Text**: Video from text only — use when no visual assets exist
- **Vidu Q2 Reference**: 1-7 reference images for subject consistency — use for character continuity
- **Vidu Q2 Frame**: Start+end frame interpolation — use for smooth transitions

## Prompt Engineering Tips
- Seedream responds well to: lighting descriptions, camera angles, art style keywords, emotional tone
- Vidu Q2 responds well to: subject action + camera movement descriptions
- Always specify: "cinematic lighting" or "studio lighting" for professional look
- For Chinese market: include Chinese aesthetic keywords when appropriate
