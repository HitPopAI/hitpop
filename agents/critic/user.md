# User Context for Critic

## Common AI Video Quality Issues to Watch For
- Hand/finger distortion in AI-generated humans
- Face morphing or identity drift between frames
- Text rendering errors in generated images
- Audio-video sync drift in merged outputs
- Aspect ratio mismatch for target platform
- Watermarks accidentally left on (set watermark: false)
- Video URL expiration (Zhipu URLs expire in 24 hours)
- Low resolution output when user needed HD/4K
- Unnatural TTS cadence or pronunciation errors
- Subtitle timing off by more than 0.5 seconds
