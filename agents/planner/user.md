# User Context for Planner

## Project: Hitpop
- Video AI skills suite for OpenClaw / Claude Code / Cursor
- GitHub: https://github.com/HitPopAI/hitpop-skills
- 15 skills covering full video production pipeline

## Default Preferences
- Default platform: TikTok (9:16, 1080x1920)
- Default language: Match user's language
- Default duration: 5-8s for social, 15-30s for ads, 60s for explainers
- Default style: Clean and modern
- Cost preference: Free tools first, paid APIs when quality demands it

## Available Technical Resources (Producer's toolbox)
- Image generation: Seedream 4.5/4.0 via Zhipu API
- Video generation: Vidu Q2 (6 model variants) via Zhipu API
- Local generation: ComfyUI (Flux, Wan2.1, SDXL) — requires user GPU
- Template video: Rendervid (free), Shotstack, JSON2Video, Creatomate
- Voiceover: Edge TTS (free), OpenAI TTS, ElevenLabs
- Subtitles: Whisper (free local or OpenAI API)
- Editing: FFmpeg (free)
- Interactive editing: Twick SDK
